# 故障處理

當自動販賣機出現故障時，快速準確的診斷和處理是確保設備正常運行的關鍵。本指南將協助您識別常見故障、執行基本排除步驟，以及了解何時需要專業技術支援。

## 故障診斷流程

### 基本診斷步驟

1. **確認故障現象**
   - 記錄具體的故障表現
   - 確認故障發生的時間和頻率
   - 檢查是否影響正常銷售

2. **查看系統狀態**
   - 登入後台系統查看設備狀態
   - 檢查最近的警報記錄
   - 查看設備運行日誌

3. **執行初步排除**
   - 嘗試遠端重啟設備
   - 檢查網路連接狀態
   - 確認電源供應正常

4. **評估故障嚴重程度**
   - 判斷是否需要立即處理
   - 評估對營運的影響程度
   - 決定處理優先級

## 常見故障類型

### 🔴 緊急故障

#### 設備完全無法運作
**症狀：**
- 設備螢幕黑屏
- 無法接受付款
- 完全無回應

**可能原因：**
- 電源故障
- 主機板故障
- 系統軟體崩潰

**處理步驟：**
1. 檢查電源連接和電源開關
2. 嘗試硬體重啟（按住電源鍵10秒）
3. 檢查保險絲是否燒毀
4. 如問題持續，立即聯絡技術支援

#### 網路連接中斷
**症狀：**
- 後台顯示設備離線
- 無法遠端控制設備
- 交易數據無法上傳

**處理步驟：**
1. 檢查網路線連接
2. 重啟網路設備（路由器、交換器）
3. 測試網路連接品質
4. 檢查防火牆設定
5. 聯絡網路服務提供商

### 🟡 重要故障

#### 商品卡住無法出貨
**症狀：**
- 客戶付款後商品未掉落
- 出貨口有商品卡住
- 螺旋推進器不轉動

**處理步驟：**
1. 遠端執行「強制出貨」指令
2. 檢查商品擺放是否正確
3. 清理出貨通道障礙物
4. 調整商品擺放方式
5. 如持續發生，安排現場維修

#### 找零系統故障
**症狀：**
- 無法找零
- 找零金額錯誤
- 硬幣卡住

**處理步驟：**
1. 檢查硬幣存量是否充足
2. 清理硬幣通道
3. 重新校準找零系統
4. 檢查硬幣識別器
5. 補充適當面額的硬幣

#### 溫度控制異常
**症狀：**
- 冷藏溫度過高或過低
- 溫度顯示異常
- 壓縮機不運轉

**處理步驟：**
1. 檢查溫度設定值
2. 清潔冷凝器和散熱器
3. 檢查冷媒是否充足
4. 確認壓縮機運行狀態
5. 聯絡專業冷凍技師

### 🔵 一般故障

#### 螢幕顯示異常
**症狀：**
- 螢幕閃爍或花屏
- 觸控不靈敏
- 顯示內容錯誤

**處理步驟：**
1. 重啟設備系統
2. 檢查螢幕連接線
3. 清潔螢幕表面
4. 重新校準觸控功能
5. 更新顯示驅動程式

#### 付款系統問題
**症狀：**
- 無法識別紙鈔或硬幣
- 信用卡讀取失敗
- 行動支付無法使用

**處理步驟：**
1. 清潔紙鈔和硬幣識別器
2. 檢查付款模組連接
3. 重啟付款系統
4. 更新付款軟體
5. 測試各種付款方式

## 遠端診斷工具

### 系統診斷功能

#### 設備健康檢查
1. 登入後台系統
2. 選擇目標設備
3. 點擊「系統診斷」
4. 執行全面健康檢查
5. 查看診斷報告

#### 遠端控制選項
- **重啟設備**：軟體重啟或硬體重啟
- **清除快取**：清理系統暫存資料
- **重置設定**：恢復預設配置
- **強制出貨**：手動觸發商品出貨
- **測試模式**：進入設備測試狀態

### 日誌分析

#### 查看系統日誌
1. 進入「設備管理」→「故障處理」
2. 選擇「系統日誌」
3. 設定查詢時間範圍
4. 篩選日誌類型：
   - 錯誤日誌
   - 警告日誌
   - 交易日誌
   - 系統日誌

#### 日誌解讀
常見錯誤代碼：
- **E001**：電源異常
- **E002**：網路連接失敗
- **E003**：商品出貨失敗
- **E004**：付款處理錯誤
- **E005**：溫度控制異常

## 維修流程

### 自助維修

#### 基本維修工具
- 螺絲起子組
- 清潔用品
- 潤滑油
- 備用保險絲
- 多功能電錶

#### 可自行處理的問題
- 清潔設備外觀和內部
- 更換燒毀的保險絲
- 調整商品擺放位置
- 補充找零硬幣
- 重新啟動設備

### 專業維修

#### 需要專業技師的情況
- 電路板故障
- 壓縮機問題
- 主要機械部件損壞
- 軟體系統崩潰
- 安全系統故障

#### 維修申請流程
1. 在後台系統提交維修申請
2. 填寫故障描述和緊急程度
3. 上傳相關照片或影片
4. 等待技師聯絡確認
5. 安排維修時間

### 預防性維護

#### 定期維護項目

**每日檢查：**
- 設備外觀清潔
- 螢幕顯示正常
- 基本功能測試

**每週檢查：**
- 清潔內部灰塵
- 檢查商品擺放
- 測試所有付款方式
- 檢查溫度控制

**每月檢查：**
- 深度清潔設備
- 檢查機械部件
- 更新軟體系統
- 校準各項功能

**每季檢查：**
- 專業技師全面檢查
- 更換易損耗零件
- 系統效能優化
- 安全功能測試

## 常見問題

### Q: 設備故障時是否會影響銷售？

**A:** 這取決於故障類型：
- 部分功能故障可能仍可正常銷售
- 嚴重故障會自動停止服務
- 系統會顯示「暫停服務」訊息

### Q: 多久會有技師到現場維修？

**A:** 回應時間根據故障嚴重程度：
- 緊急故障：4小時內
- 重要故障：24小時內
- 一般故障：48小時內

### Q: 如何預防設備故障？

**A:** 建議採取以下預防措施：
- 定期執行預防性維護
- 保持設備清潔
- 及時處理小問題
- 定期更新軟體系統
