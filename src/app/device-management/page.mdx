# 設備管理

全面管理您的 AMIGO 自動販賣機設備，包括設備狀態監控、配置管理、故障診斷等功能，確保設備正常運行和最佳性能。

## 功能概覽

設備管理模塊提供完整的設備生命週期管理功能，幫助您：

- **即時監控**：24/7 監控設備運行狀態
- **預防維護**：提前發現潛在問題
- **快速響應**：及時處理設備故障
- **性能優化**：持續改善設備表現

## 主要功能

### 設備列表管理

查看和管理所有設備的基本資訊：

- 設備編號和名稱
- 設備型號和規格
- 安裝位置和地址
- 設備狀態和最後更新時間

## 設備狀態說明

### 正常運行 🟢
設備運行正常，所有系統功能正常。

### 警告狀態 🟡
設備運行基本正常，但存在需要關注的問題：
- 庫存不足
- 溫度異常但在可接受範圍內
- 網路連線不穩定

### 故障狀態 🔴
設備無法正常提供服務：
- 硬體故障
- 系統錯誤
- 嚴重溫度異常
- 網路完全中斷

### 維護中 🔵
設備正在進行計劃性維護或修理。

## 操作權限

不同角色的操作權限：

### 子代理商
- 查看設備狀態
- 接收故障通知
- 申請維修服務
- 查看維修記錄

### 技術人員
- 所有子代理商權限
- 遠程診斷設備
- 更新設備配置
- 執行維護操作

### 系統管理員
- 所有權限
- 添加/移除設備
- 管理用戶權限
- 系統配置管理

## 最佳實踐

### 日常監控
- 每日檢查設備狀態總覽
- 關注溫度和庫存警報
- 定期查看設備性能報告

### 預防維護
- 按時執行計劃性維護
- 及時補充庫存
- 保持設備清潔

### 故障處理
- 收到警報後立即查看詳情
- 按照故障處理流程操作
- 記錄問題和解決方案

## 常見問題

<Note>
  如果設備出現故障，請先查看故障診斷頁面的常見問題解決方案。如無法解決，請立即聯絡技術支援。
</Note>

### 設備離線怎麼辦？
1. 檢查設備電源是否正常
2. 確認網路連線狀態
3. 重啟設備網路模組
4. 聯絡技術支援

### 溫度異常如何處理？
1. 檢查設備周圍環境
2. 確認冷卻系統運行狀態
3. 調整溫度設定參數
4. 必要時申請維修

## 技術支援

如需技術支援，請：

1. 記錄設備編號和故障現象
2. 截圖相關錯誤信息
3. 通過系統提交維修申請
4. 或直接撥打技術支援熱線
