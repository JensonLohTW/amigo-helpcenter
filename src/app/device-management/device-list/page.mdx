# 設備列表

設備列表是 AMIGO 後台系統的核心功能之一，讓您能夠集中管理所有自動販賣機設備。透過設備列表，您可以快速查看設備狀態、位置資訊、運行數據等重要資訊。

## 功能概述

設備列表提供以下主要功能：

- **設備狀態監控**：即時查看所有設備的運行狀態
- **設備資訊管理**：查看和編輯設備的基本資訊
- **批量操作**：對多個設備執行批量操作
- **篩選和搜索**：快速找到特定設備
- **設備分組**：按地區、類型等條件對設備進行分組

## 查看設備列表

### 進入設備列表

1. 登入 AMIGO 後台系統
2. 在主選單中點擊「設備管理」
3. 選擇「設備列表」進入設備管理頁面

### 設備列表介面說明

設備列表頁面包含以下主要區域：

#### 頂部工具欄
- **搜索框**：輸入設備名稱、編號或位置進行搜索
- **篩選器**：按狀態、地區、類型等條件篩選設備
- **新增設備**：添加新的自動販賣機設備
- **批量操作**：對選中的設備執行批量操作

#### 設備列表表格
每個設備顯示以下資訊：

| 欄位 | 說明 |
|------|------|
| 設備編號 | 唯一識別碼，用於設備管理 |
| 設備名稱 | 自定義的設備名稱 |
| 狀態 | 🟢 正常 / 🟡 警告 / 🔴 故障 / ⚫ 離線 |
| 位置 | 設備安裝地點 |
| 類型 | 設備型號和規格 |
| 最後更新 | 最後一次數據更新時間 |
| 操作 | 查看詳情、編輯、重啟等操作 |

## 設備操作

### 查看設備詳情

1. 在設備列表中找到目標設備
2. 點擊設備名稱或「查看詳情」按鈕
3. 在彈出的詳情頁面中查看：
   - 設備基本資訊
   - 即時運行狀態
   - 商品庫存情況
   - 近期交易記錄
   - 維護記錄

### 編輯設備資訊

1. 點擊設備列表中的「編輯」按鈕
2. 在編輯表單中修改以下資訊：
   - 設備名稱
   - 安裝位置
   - 聯絡資訊
   - 營業時間
   - 特殊設定
3. 點擊「儲存」確認修改

### 設備控制操作

#### 重啟設備
1. 選擇需要重啟的設備
2. 點擊「操作」選單中的「重啟設備」
3. 確認重啟操作
4. 等待設備重新上線（通常需要 2-3 分鐘）

#### 鎖定/解鎖設備
- **鎖定設備**：暫停設備服務，停止銷售
- **解鎖設備**：恢復設備正常運行

#### 更新設備軟體
1. 選擇需要更新的設備
2. 點擊「軟體更新」
3. 選擇更新版本
4. 確認更新並等待完成

## 篩選和搜索

### 使用搜索功能

在搜索框中輸入以下任一資訊：
- 設備編號（如：AMG001）
- 設備名稱（如：台北車站A區）
- 位置關鍵字（如：台北、車站）

### 使用篩選器

點擊「篩選」按鈕，可按以下條件篩選：

#### 設備狀態
- ✅ 正常運行
- ⚠️ 需要注意
- ❌ 故障設備
- 📴 離線設備

#### 地區分類
- 北部地區
- 中部地區
- 南部地區
- 東部地區

#### 設備類型
- 標準型販賣機
- 冷藏型販賣機
- 熱飲型販賣機
- 複合型販賣機

## 批量操作

### 選擇多個設備

1. 勾選設備列表左側的核取方塊
2. 或點擊「全選」選擇當前頁面所有設備
3. 已選設備數量會顯示在頂部

### 批量操作選項

- **批量重啟**：同時重啟多個設備
- **批量更新**：為多個設備安裝軟體更新
- **批量設定**：統一修改設備參數
- **匯出資料**：將選中設備的資料匯出為 Excel 檔案

## 設備狀態說明

### 狀態指示器

| 狀態 | 圖示 | 說明 | 建議操作 |
|------|------|------|----------|
| 正常 | 🟢 | 設備運行正常，無異常 | 定期檢查即可 |
| 警告 | 🟡 | 設備有輕微問題或需要注意 | 查看詳情，及時處理 |
| 故障 | 🔴 | 設備發生故障，無法正常服務 | 立即檢修或聯絡技術支援 |
| 離線 | ⚫ | 設備失去網路連接 | 檢查網路連接或現場檢查 |

### 常見狀態問題

#### 設備顯示離線
**可能原因：**
- 網路連接中斷
- 設備電源問題
- 系統軟體異常

**解決方法：**
1. 檢查設備網路連接
2. 確認電源供應正常
3. 嘗試遠端重啟設備
4. 如問題持續，請聯絡現場維護人員

#### 設備狀態為警告
**常見警告類型：**
- 商品庫存不足
- 找零硬幣不足
- 溫度異常
- 門鎖異常

**處理方式：**
1. 查看具體警告內容
2. 根據警告類型採取相應措施
3. 安排補貨或維護
4. 確認問題解決後狀態會自動更新

## 常見問題

### Q: 為什麼有些設備無法顯示即時狀態？

**A:** 可能的原因包括：
- 設備網路連接不穩定
- 設備軟體版本過舊
- 系統正在維護中

建議檢查設備網路連接，或聯絡技術支援協助。

### Q: 如何快速找到特定地區的設備？

**A:** 您可以：
1. 使用搜索功能輸入地區名稱
2. 使用篩選器選擇特定地區
3. 在設備列表中按位置欄位排序

### Q: 批量操作失敗怎麼辦？

**A:** 如果批量操作失敗：
1. 檢查網路連接是否穩定
2. 確認所選設備都處於可操作狀態
3. 嘗試分批次執行操作
4. 查看操作日誌了解具體失敗原因
