# 常見問題

AMIGO 自動販賣機後台系統常見問題解答，涵蓋系統操作、設備管理、商品管理、訂單處理等各方面的問題和解決方案。

## 系統操作問題

### 登入相關

**Q: 忘記密碼怎麼辦？**
A: 在登入頁面點擊「忘記密碼」，輸入您的帳號，系統會發送重置密碼的郵件到您的註冊信箱。

**Q: 為什麼無法登入系統？**
A: 可能的原因包括：
- 帳號或密碼錯誤
- 帳號被鎖定（多次登入失敗）
- 網路連線問題
- 瀏覽器快取問題

建議清除瀏覽器快取後重試，或聯絡技術支援。

**Q: 系統會自動登出嗎？**
A: 是的，為了安全考量，系統會在 30 分鐘無操作後自動登出。

### 介面操作

**Q: 如何使用搜索功能？**
A: 點擊頂部搜索框或使用快捷鍵 ⌘K（Mac）/ Ctrl+K（Windows）開啟搜索，輸入關鍵詞即可搜索相關內容。

**Q: 如何切換深色/淺色主題？**
A: 點擊頂部導航欄的主題切換按鈕（月亮/太陽圖標）即可切換主題。

**Q: 手機可以使用這個系統嗎？**
A: 可以，系統採用響應式設計，完全支援手機和平板設備訪問。

## 設備管理問題

### 設備狀態

**Q: 設備顯示離線怎麼辦？**
A: 請按以下步驟檢查：
1. 確認設備電源是否正常
2. 檢查網路連線狀態
3. 重啟設備的網路模組
4. 如問題持續，請聯絡技術支援

**Q: 設備溫度異常如何處理？**
A: 
1. 檢查設備周圍是否有遮擋物
2. 確認冷卻系統是否正常運行
3. 檢查設備內部是否需要清潔
4. 如溫度持續異常，請立即聯絡維修人員

**Q: 如何查看設備的維修記錄？**
A: 進入「設備管理」→「設備列表」，點擊特定設備，在設備詳情頁面可查看完整的維修歷史記錄。

### 設備維護

**Q: 設備多久需要維護一次？**
A: 建議每月進行一次基礎維護，每季度進行一次深度維護。系統會自動提醒維護時間。

**Q: 可以自己維護設備嗎？**
A: 基礎清潔和簡單檢查可以自行進行，但涉及硬體維修的問題建議聯絡專業技術人員。

## 商品管理問題

### 商品上架

**Q: 如何添加新商品？**
A: 進入「商品管理」→「庫存管理」，點擊「添加商品」，填寫商品資訊後保存即可。

**Q: 商品圖片有什麼要求？**
A: 建議使用：
- 格式：JPG 或 PNG
- 尺寸：建議 800x600 像素
- 大小：不超過 2MB
- 背景：建議使用白色背景

**Q: 如何批量修改商品價格？**
A: 在「商品管理」→「價格管理」中，選擇多個商品，點擊「批量操作」→「批量調價」即可。

### 庫存管理

**Q: 庫存數據不準確怎麼辦？**
A: 可能原因：
- 設備感應器故障
- 人工補貨未及時更新系統
- 系統同步延遲

建議進行庫存盤點，手動調整庫存數據。

**Q: 如何設定庫存警報？**
A: 在商品編輯頁面設定「安全庫存」數量，當庫存低於此數量時系統會自動發送警報。

**Q: 庫存為零但設備還在出貨怎麼辦？**
A: 這可能是設備感應器故障，請立即：
1. 暫停該商品的銷售
2. 檢查設備感應器
3. 聯絡技術支援進行維修

## 訂單管理問題

### 訂單處理

**Q: 客戶反映支付成功但沒收到商品怎麼辦？**
A: 
1. 在「訂單管理」中查找該筆訂單
2. 檢查訂單狀態和設備日誌
3. 如確認是設備故障，立即處理退款
4. 記錄問題並安排設備檢修

**Q: 如何處理重複扣款的訂單？**
A: 
1. 確認是否真的重複扣款
2. 查看訂單詳情和支付記錄
3. 聯絡支付平台確認
4. 為客戶處理退款

**Q: 訂單數據可以導出嗎？**
A: 可以，在「訂單管理」頁面點擊「導出」按鈕，選擇時間範圍和格式即可導出訂單數據。

### 退款處理

**Q: 退款需要多長時間到帳？**
A: 
- 電子支付：1-3 個工作日
- 信用卡：3-7 個工作日
- 現金：需要現場處理

**Q: 如何查看退款狀態？**
A: 在「訂單管理」→「退款管理」中可以查看所有退款申請的處理狀態。

## 財務報表問題

### 數據查詢

**Q: 為什麼財務數據和實際不符？**
A: 可能原因：
- 數據同步延遲
- 時區設定問題
- 篩選條件設定錯誤
- 系統計算錯誤

建議檢查篩選條件和時間範圍設定。

**Q: 如何查看特定時期的財務報表？**
A: 在財務報表頁面使用日期篩選器，選擇您需要的時間範圍即可。

**Q: 報表可以自動發送嗎？**
A: 可以，在「系統設定」→「通知設定」中配置定期報表推送。

### 數據分析

**Q: 如何分析商品的盈利能力？**
A: 在「財務報表」→「利潤分析」中可以查看各商品的毛利率、淨利率等指標。

**Q: 成本包括哪些項目？**
A: 系統計算的成本包括：
- 商品採購成本
- 設備租金和維護費用
- 電費和其他運營成本
- 人力成本（如有）

## 技術問題

### 系統性能

**Q: 系統運行很慢怎麼辦？**
A: 
1. 檢查網路連線速度
2. 清除瀏覽器快取
3. 關閉其他不必要的瀏覽器標籤
4. 嘗試使用其他瀏覽器

**Q: 頁面無法載入怎麼辦？**
A: 
1. 重新整理頁面
2. 檢查網路連線
3. 清除瀏覽器快取和 Cookie
4. 嘗試無痕模式瀏覽

### 瀏覽器相容性

**Q: 支援哪些瀏覽器？**
A: 系統支援以下瀏覽器的最新版本：
- Chrome（推薦）
- Firefox
- Safari
- Edge

**Q: 為什麼某些功能無法使用？**
A: 可能是瀏覽器版本過舊或禁用了 JavaScript，請：
1. 更新瀏覽器到最新版本
2. 確保啟用 JavaScript
3. 檢查瀏覽器安全設定

## 帳戶和權限問題

**Q: 如何修改個人資訊？**
A: 進入「系統設定」→「帳戶設定」，在個人資訊頁面進行修改。

**Q: 為什麼無法訪問某些功能？**
A: 這可能是權限限制，請聯絡系統管理員檢查您的帳戶權限設定。

**Q: 如何申請更高的權限？**
A: 請聯絡您的上級主管或系統管理員提出權限申請。
