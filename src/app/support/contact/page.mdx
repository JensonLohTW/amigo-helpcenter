# 聯絡我們

聯絡 AMIGO 技術支援團隊，我們提供多種聯絡方式，包括電話、郵件、線上客服等，為您提供專業的技術支援服務。

## 聯絡方式

### 24/7 緊急技術支援熱線 🚨

**電話：0800-123-456**

- **服務時間**：24 小時全天候服務
- **適用情況**：系統故障、設備緊急故障、安全問題
- **響應時間**：30 分鐘內回應
- **服務語言**：中文、英文

<Note>
  緊急熱線僅用於緊急情況。一般問題請使用其他聯絡方式，以確保緊急熱線暢通。
</Note>

- **服務語言**：中文

## 工單系統

### 提交工單

通過工單系統提交詳細的技術問題：

1. **登入系統**：使用您的帳戶登入後台系統
2. **進入工單系統**：點擊「技術支援」→「提交工單」
3. **填寫工單資訊**：
   - 問題類型
   - 優先級別
   - 詳細描述
   - 相關截圖
4. **提交工單**：系統會自動分配工單編號

### 工單追蹤

- **工單狀態**：已提交、處理中、等待回覆、已解決、已關閉
- **進度通知**：工單狀態變更時會自動發送通知
- **歷史記錄**：可查看所有歷史工單記錄

## 聯絡前的準備

### 問題描述準備

為了更快速地解決您的問題，請準備以下資訊：

1. **問題詳細描述**
   - 具體的問題現象
   - 問題發生的時間
   - 問題發生的頻率

2. **重現步驟**
   - 詳細的操作步驟
   - 預期結果 vs 實際結果
   - 是否能穩定重現

3. **環境資訊**
   - 使用的瀏覽器和版本
   - 作業系統
   - 網路環境
   - 設備型號（如涉及設備問題）

4. **錯誤資訊**
   - 完整的錯誤訊息
   - 錯誤代碼
   - 相關截圖或錄屏

### 帳戶資訊

請準備以下帳戶相關資訊：

- **帳戶名稱**
- **公司/組織名稱**
- **聯絡人姓名**
- **聯絡電話**
- **電子郵箱**

## 問題分類指南

### 緊急問題 🆘
**立即撥打緊急熱線：0800-123-456**

- 系統完全無法訪問
- 設備大面積故障
- 資料安全問題
- 支付系統故障

### 高優先級問題 🔴
**建議使用電話或線上客服**

- 重要功能無法使用
- 設備部分故障
- 訂單處理異常
- 財務數據錯誤

### 中優先級問題 🟡
**建議使用郵件或工單系統**

- 部分功能異常
- 操作不便
- 報表生成問題
- 介面顯示問題

### 低優先級問題 🟢
**建議使用郵件或工單系統**

- 功能改進建議
- 使用諮詢
- 培訓需求
- 文檔問題
