# 庫存管理

庫存管理是確保自動販賣機正常營運的核心功能。透過有效的庫存管理，您可以避免缺貨情況、優化補貨效率，並最大化銷售收益。本指南將協助您掌握庫存管理的各項功能。

## 功能概述

庫存管理系統提供以下主要功能：

- **即時庫存監控**：查看所有設備的即時庫存狀況
- **自動補貨提醒**：基於銷售數據的智能補貨建議
- **庫存預警系統**：低庫存和缺貨自動警報
- **補貨計劃管理**：制定和執行補貨計劃
- **庫存分析報告**：庫存週轉率和效率分析
- **批量庫存操作**：多設備庫存統一管理

## 庫存監控

### 查看庫存狀況

#### 進入庫存管理
1. 登入 AMIGO 後台系統
2. 點擊「商品管理」選單
3. 選擇「庫存管理」進入庫存監控頁面

#### 庫存總覽儀表板
庫存總覽顯示以下關鍵資訊：

- **總庫存價值**：所有設備庫存的總價值
- **低庫存商品數**：需要補貨的商品種類
- **缺貨設備數**：有商品缺貨的設備數量
- **平均庫存天數**：基於銷售速度的庫存可維持天數

#### 設備庫存列表
每個設備顯示以下庫存資訊：

| 欄位 | 說明 |
|------|------|
| 設備名稱 | 設備識別名稱和位置 |
| 庫存狀態 | 🟢 充足 / 🟡 偏低 / 🔴 缺貨 |
| 商品種類 | 設備內商品種類數量 |
| 總庫存量 | 所有商品的總數量 |
| 預估可售天數 | 基於銷售速度的預估 |
| 最後補貨時間 | 上次補貨的日期時間 |

### 商品庫存詳情

#### 查看單一設備庫存
1. 在設備列表中點擊目標設備
2. 查看該設備的詳細庫存資訊：
   - 每個商品的剩餘數量
   - 商品在貨道中的位置
   - 每日平均銷售量
   - 預估缺貨時間

#### 商品庫存狀態
- **🟢 庫存充足**：剩餘量 > 安全庫存量
- **🟡 庫存偏低**：剩餘量 ≤ 安全庫存量
- **🔴 即將缺貨**：剩餘量 ≤ 最低庫存量
- **⚫ 已缺貨**：剩餘量 = 0

## 補貨管理

### 補貨提醒系統

#### 自動補貨提醒
系統會根據以下條件自動發送補貨提醒：
- 商品庫存低於安全庫存量
- 預估在設定天數內會缺貨
- 熱銷商品庫存快速下降
- 特殊活動期間庫存需求增加

#### 補貨提醒設定
1. 進入「庫存設定」頁面
2. 為每個商品設定：
   - **安全庫存量**：觸發補貨提醒的數量
   - **最低庫存量**：緊急補貨的數量
   - **最大庫存量**：單次補貨的上限
   - **提醒提前天數**：提前幾天發送提醒

### 補貨計劃

#### 創建補貨計劃
1. 點擊「新增補貨計劃」
2. 選擇需要補貨的設備
3. 設定補貨商品和數量：
   - 根據系統建議調整數量
   - 考慮季節性和促銷活動
   - 確認貨道容量限制
4. 設定補貨時間和負責人
5. 儲存補貨計劃

#### 補貨建議算法
系統基於以下因素提供補貨建議：
- **歷史銷售數據**：過去30天的平均銷售量
- **季節性趨勢**：同期歷史數據比較
- **庫存週轉率**：商品流動速度分析
- **設備容量**：貨道最大容量限制
- **供應商交期**：商品供應時間考量

### 執行補貨作業

#### 補貨作業流程
1. **準備階段**
   - 確認補貨計劃和商品清單
   - 準備所需商品和工具
   - 檢查設備狀態和安全性

2. **執行補貨**
   - 開啟設備並進入維護模式
   - 按照貨道配置補充商品
   - 確認商品擺放正確且穩固
   - 檢查商品有效期限

3. **完成作業**
   - 在系統中記錄補貨數量
   - 拍照記錄補貨後狀態
   - 測試設備出貨功能
   - 恢復設備正常營運

#### 補貨記錄管理
每次補貨都應記錄：
- 補貨日期和時間
- 補貨人員資訊
- 補貨商品和數量
- 補貨前後庫存對比
- 異常情況說明

## 庫存預警

### 預警類型

#### 低庫存預警 🟡
- **觸發條件**：庫存量低於安全庫存量
- **處理建議**：安排常規補貨
- **緊急程度**：中等

#### 缺貨預警 🔴
- **觸發條件**：商品完全售完
- **處理建議**：立即安排緊急補貨
- **緊急程度**：高

#### 滯銷預警 🔵
- **觸發條件**：商品超過設定天數未售出
- **處理建議**：考慮調整價格或更換商品
- **緊急程度**：低

#### 過期預警 ⚠️
- **觸發條件**：商品接近有效期限
- **處理建議**：優先銷售或下架處理
- **緊急程度**：高

### 預警設定

#### 預警參數設定
1. 進入「預警設定」頁面
2. 為不同商品類型設定：
   - 安全庫存天數（建議7-14天）
   - 滯銷判定天數（建議30天）
   - 過期提醒天數（建議7天）
   - 預警通知方式

#### 通知設定
- **即時通知**：APP推播、簡訊
- **每日報告**：電子郵件彙總
- **週報**：詳細分析報告
- **緊急通知**：電話語音提醒

## 庫存分析

### 庫存報表

#### 庫存狀況報表
- **即時庫存報表**：當前所有設備庫存狀況
- **庫存變動報表**：特定期間庫存變化
- **補貨記錄報表**：補貨作業歷史記錄
- **缺貨統計報表**：缺貨頻率和影響分析

#### 庫存效率分析
- **庫存週轉率**：庫存流動效率指標
- **平均庫存天數**：商品平均停留時間
- **缺貨率**：缺貨時間占比
- **補貨頻率**：平均補貨間隔時間

### 數據視覺化

#### 圖表類型
- **庫存趨勢圖**：顯示庫存量變化趨勢
- **補貨頻率圖**：各設備補貨頻率比較
- **商品週轉圖**：不同商品的週轉率
- **缺貨影響圖**：缺貨對銷售的影響

#### 自訂報表
1. 選擇報表類型和時間範圍
2. 選擇要分析的設備和商品
3. 設定分析維度和指標
4. 生成報表並匯出

## 庫存優化建議

### 庫存策略

#### ABC分析法
根據銷售額和銷量將商品分類：
- **A類商品**：高銷售額，需保持充足庫存
- **B類商品**：中等銷售額，適中庫存管理
- **C類商品**：低銷售額，最小庫存策略

#### 季節性調整
- **夏季**：增加冷飲、冰品庫存
- **冬季**：增加熱飲、暖食庫存
- **節慶期間**：調整特殊商品庫存
- **學期變化**：校園設備庫存調整

### 成本控制

#### 庫存成本分析
- **持有成本**：庫存佔用資金成本
- **缺貨成本**：失去銷售機會成本
- **補貨成本**：運輸和人工成本
- **過期成本**：商品過期損失

#### 優化建議
- 根據銷售數據調整安全庫存量
- 優化補貨頻率降低運輸成本
- 實施先進先出原則減少過期
- 定期檢討滯銷商品策略

## 常見問題

### Q: 如何設定合適的安全庫存量？

**A:** 建議考慮以下因素：
- 商品的平均日銷量
- 供應商的交貨週期
- 銷售的季節性變化
- 設備的地理位置和客流量
一般建議設定為7-14天的銷售量

### Q: 系統如何計算補貨建議？

**A:** 系統綜合考慮：
- 過去30天的銷售趨勢
- 當前庫存量和安全庫存
- 貨道最大容量限制
- 商品的保存期限
- 即將到來的促銷活動

### Q: 如何處理滯銷商品？

**A:** 建議採取以下措施：
1. 分析滯銷原因（價格、位置、季節等）
2. 考慮調整售價或促銷
3. 更換為熱銷商品
4. 調整商品在貨道中的位置
5. 分析客戶偏好調整商品組合
