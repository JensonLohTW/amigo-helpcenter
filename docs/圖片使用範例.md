# 圖片使用範例

本文檔提供各種圖片使用場景的具體範例。

## 📸 基本圖片使用

### 單張圖片
```jsx
<Image 
  src="pages/product-management/dashboard.png"
  alt="商品管理儀表板"
  size="lg"
  style={['rounded', 'shadow']}
  caption="商品管理主要介面，顯示庫存狀態和銷售數據"
/>
```

### 小圖標
```jsx
<Image 
  src="common/icon-warning.png"
  alt="警告圖標"
  size="xs"
  style="rounded-full"
/>
```

### 品牌標誌
```jsx
<Image 
  src="logos/amigo-logo.svg"
  alt="AMIGO 標誌"
  size="sm"
  priority={true}
/>
```

## 🔧 進階使用

### 使用工具函數
```jsx
import { img, ImagePathBuilder } from '@/lib/images'

// 簡潔的函數調用
<Image src={img.page('product-management', 'dashboard.png')} alt="儀表板" />
<Image src={img.common('success-icon.png')} alt="成功" />
<Image src={img.logo('partner-logo.svg')} alt="合作夥伴" />

// 鏈式構建器
<Image 
  src={ImagePathBuilder.page('device-management').file('device-status.png')} 
  alt="設備狀態" 
/>

<Image 
  src={ImagePathBuilder.feature('analytics').file('sales-chart.png')} 
  alt="銷售圖表" 
/>
```

### 條件渲染
```jsx
{isLoading ? (
  <div className="animate-pulse bg-gray-200 rounded h-48 w-full" />
) : (
  <Image 
    src="pages/dashboard/main-view.png"
    alt="主要視圖"
    size="lg"
    priority={true}
  />
)}
```

### 互動式圖片
```jsx
<Image 
  src="ui/interactive-demo.png"
  alt="互動式示範"
  size="lg"
  onClick={() => setShowDemo(true)}

/>
```

## 📊 實際應用場景

### 商品管理頁面
```jsx
export default function ProductManagementPage() {
  return (
    <div>
      <h1>商品管理</h1>
      
      {/* 主要功能展示 */}
      <ImageGallery
        images={[
          { src: img.page('product-management', 'inventory.png'), alt: '庫存管理' },
          { src: img.page('product-management', 'pricing.png'), alt: '價格管理' },
          { src: img.page('product-management', 'categories.png'), alt: '分類管理' },
        ]}
        columns={3}
      />
      
      {/* 操作流程說明 */}
      <ImageWithSteps
        image={{ src: img.page('product-management', 'workflow.png'), alt: '工作流程' }}
        steps={['新增商品', '設定價格', '分配庫存', '上架銷售']}
      />
    </div>
  )
}
```
